How it works

Three steps. No friction.

Works with any app — browser, editor, email, document, terminal.

STEP 01

Select any text

Highlight text in any application on your screen.

STEP 02

Press Ctrl+C twice

Skuoty captures your selection and opens instantly.

STEP 03

Get your result

Pick a plugin, run it, paste the result back in one click.

Features

Everything you need

Powerful by default. Zero complexity.

Instant trigger

Activated by double Ctrl+C from any app. No window switching, no context loss.

Plugin system

4 built-in plugins, fully customizable. Add your own with a simple JSON definition — no code required.

5 AI providers

Gemini, OpenAI, Anthropic, OpenRouter and local Ollama. Your keys, your models, your data.

Encrypted sessions

Settings and API keys encrypted with AES-256-GCM + PBKDF2. Password-protected, never stored in plain text.

Multilingual UI

Interface in English, Italian, Spanish, French and German. More via community contributions.

Export & import

Back up your entire configuration encrypted and portable. Restore it on any machine in seconds.

Plugins

Ready from day one

4 built-in plugins. Fully customizable, removable, and extensible.

Translate

8 target languages: English, Italian, Spanish, French, German, Portuguese, Japanese, Chinese.

Summarize

3 detail levels: short (2–3 sentences), medium paragraph, or detailed with bullet points.

Rewrite

7 tones: Friendly, Professional, Concise, Persuasive, Simple, Assertive, Empathetic.

Fix Grammar

Corrects grammar, spelling, punctuation, verb tenses and word choice, preserving your original style.

Need something custom? Write your own plugin in JSON — no code required.

AI Providers

Your AI, your choice

Connect to any supported provider with your own API key. Or run fully offline with Ollama — no internet, no cost.

Gemini OpenAI Anthropic OpenRouter Ollama — local

API keys are encrypted on your device and sent exclusively to the provider you configure.

Free and open source

Skuoty is MIT licensed. Inspect the code, contribute, fork it, build on it. No telemetry, no tracking, no subscriptions. Your data stays on your machine.
